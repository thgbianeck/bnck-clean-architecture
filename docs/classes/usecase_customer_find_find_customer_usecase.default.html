Class default

A classe FindCustomerUseCase é responsável por executar o caso de uso de busca de cliente. Ele utiliza um CustomerRepositoryInterface para recuperar dados de um cliente pelo seu ID.

Remarks

O caso de uso recebe um InputFindCustomerDto contendo o ID do cliente a ser buscado. Ele retorna uma promessa que resolve para um OutputFindCustomerDto contendo os detalhes do cliente. Se o cliente não for encontrado, retorna null.

Example

const customerRepository = new CustomerRepository();
const findCustomerUseCase = new FindCustomerUseCase(customerRepository);
const input: InputFindCustomerDto = { id: 123 };
const output = await findCustomerUseCase.execute(input);
console.log(output); // Saída: { id: 123, name: 'John Doe', address: { street: '123 Main St', city: 'New York', number: '100', zip: '10001' } }
